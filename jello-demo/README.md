# jello-demo

## 安装 fis3

```
$ npm install fis3 -g
```

**fis3 版本必须 >= 3.2.1** 目前请通过 `npm install -g fis3@3.2.1` 安装。

## 初始化

```bash
$ mkdir demo
$ cd demo
$ fis3 init jello-demo
```

## 运行 & 预览

```bash
$ fis3 release
$ fis3 server start
```

## 发布产品代码

开启压缩和合并等等。。

```bash
$ fis3 release prod -d /path/of/your/j2ee/app
```

更多信息请访问：https://github.com/fex-team/fis3-jello
